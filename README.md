# SaaS Project - Next.js + Django

This is a project-based learning SaaS application built with a professional full-stack architecture.

## Stack

- Next.js + TypeScript frontend
- Django + Django REST Framework backend
- PostgreSQL database later
- Docker and Docker Compose
- Git and GitHub
- GitHub Actions CI/CD later

## Project Structure

```text
saas-project-next-django/
├── frontend-nextjs/        # Next.js frontend application
├── backend-django-api/     # Django REST API backend
├── infrastructure-docker/  # Docker, PostgreSQL, Nginx, deployment files later
└── README.md
```

## Learning Goal

Build a real SaaS website step by step using clean code, standard professional methods, testing, security, deployment, and SEO practices.
